Welcome to My Blog #

Hey there! I’m Hao Lou (also known as Phyllis), and this is the very first post on my freshly built personal site.

After procrastinating for way too long, I finally sat down and forked Anthony Fu’s elegant site template to create my own little corner of the internet. The design is minimal, fast, and distraction-free — exactly how I like my code and my thoughts.

Why Start a Blog? #

I believe writing is the best way to clarify thinking. As a Computer Science student at Shanghai Jiao Tong University, I spend most of my days exploring new technologies, building side projects, and chasing the excitement of making something work for the first time. Putting ideas into words helps me find gaps in my understanding — and hopefully helps others who stumble upon these posts.

What I’ll Write About #

Here’s a rough map of the territory I plan to explore:

Innovative Technology #

  • Systems programming with C++ and Rust
  • Rapid prototyping and AI experiments with Python
  • Game development adventures with Godot
  • AI-assisted development with Claude Code and Vibe Coding
  • Building tools and automations with OpenClaw

Cool Projects #

  • Side project showcases — the fun, weird, and useful things I build in my free time
  • Hackathon recaps — the 48-hour sprints, the sleep deprivation, and the surprisingly good results
  • Tool-making — small utilities that solve my own problems (and maybe yours too)
  • Open source adventures — contributing to projects, maintaining my own, and learning from the community

AI & Machine Learning #

  • Practical ML — from feature engineering to model deployment
  • AI in everyday tools — using LLMs and deep learning to build things people actually use
  • Learning notes — because ML papers are dense and I need to take notes somewhere

Random Thoughts #

  • Book recommendations
  • Travel notes

The Goal #

I don’t promise a rigid posting schedule. Instead, I’ll write when I have something worth saying — whether that’s a tutorial on a shiny new framework, a deep dive into a fun project, or just a story about why my experimental build failed at 3 AM.

Quality over quantity. Curiosity over clout.
